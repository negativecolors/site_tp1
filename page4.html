développer back end



image:

définition:

le développer back end est la personne qui se charge du code,donc les éléments technique et du fonctionnement.Il gère toute la partie non visible du site internet

communication:

Il communique avec le front end pour pouvoir coder le site web et le développer mobile pour bien ajuster sur téléphone.

les missions:

Ses missions sont de analyser des données et échanger avec une base de données,collaborer avec les développeurs front-end et mobile pour concevoir les codes les plus cohérents pour l’expérience de l’utilisateur et enfin établir la bonne communication entre les développeurs front-end et mobile, et la base de données

compétence back end:

il doit savoir utiliser PHP, Ruby, Python et Java
