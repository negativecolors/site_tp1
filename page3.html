développer front end

image:

définition:

le développer front end est la personne qui se charge du visuelle d'un site web et se chage aussi des interaction et de l'expèrience d'un utilisateur pour que se soit ergonomique et facile sur une plateforme.

communication:

le front end utilise l'idée finale du designer et l'intègre dans le site web. Puis se met en relation avec le back end pour terminer le code.

les missions:

Ses missions sont de transformer une maquette en code, bien faire fonctionner le code, que le site web soit en haut des recherches web, améliorer l'expèrience du client, opptimiser les pages web et enfin tester et corriger si besoin

compétence front end:

il doit savoir utiliser HTML, CSS et JavaScript
